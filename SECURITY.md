# Security Policy

Thanks for helping keep Charmap2 safe.

## Reporting a vulnerability

Please do not open a public GitHub issue for suspected security vulnerabilities.

Instead, report vulnerabilities through one of these channels:

- GitHub Security Advisories for this repository, if enabled
- direct contact with the maintainer via the contact information listed on the repository profile

Please include:

- a description of the issue
- steps to reproduce
- potential impact
- any suggested mitigation, if known

## Response goals

This project is maintained on a best-effort basis. We will try to acknowledge reports promptly, assess impact, and provide a fix or mitigation when appropriate.

## Scope notes

Charmap2 is a static site with no backend service, but client-side security concerns still matter. Reports about XSS, unsafe copy behavior, dependency vulnerabilities, supply-chain risk in build tooling, or malicious data processing are all in scope.
